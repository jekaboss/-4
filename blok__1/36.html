Завантаження на сервер через FTP

В стартовому шаблоні "Чертоги Фрілансера" 4 (ЧФ4) є можливість автоматичного завантаження зібраного проєкту (папка dist) на сервер за допомогою FTP

Для цього вкажіть необхідні дані в налаштуваннях ЧФ4 в розділі ftp:

spa - Налаштування відправлення кінцевих файлів (з папки dist) на сервер (хостинг) по FTP в режимі продакшена

host - Ім'я FTP хосту

port - FTP Порт

remoteDir - Віддалена папка на сервері. Можна використовувати змінну projectName яка містить ім'я папки проєкту

user - Ім'я користувача FTP

password - Пароль користувача FTP

Після налаштування ви можете користуватись командою термінала npm run deploy, яка об'єднує в собі команду збірки проєкту (режим продакшн) а також завантаження результату на сервер за допомогою FTP
