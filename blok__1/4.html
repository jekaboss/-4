Налаштування збірки «Чертоги Фрілансера» 4

ЧФ4 побудована так, щоб якомога більше можливостей можна було налаштувати під потреби користувача.

Для того щоб налаштувати збірку відкриваємо файл template.config.js в папці проєкту.

Перелік налаштувань:

1. vscode - Налаштування редактора VS Code

1.1 settings - Автоматичне налаштування псевдонімів для проєкту (плагін Path Autocomplete)

1.2 snippets - Автоматичне додавання сніпетів ЧФ4 в проєкт

2. navpanel - Налаштування навігаційної панелі

2.1 dev - У режимі розробника. Вмикач: true/false

2.2 build - У режимі продакшена. Вмикач: true/false

2.3 position - Позиція виводу панелі: left/right

2.4 color - Колір тексту та посилань

2.5 background - Колір фону

2.6 transition - Швидкість анімації відкриття у мілісекундах

3. server - Налаштування локального серверу

3.1 path - Налаштування шляхів до файлів: ./ - відносний, / - абсолютний

3.2 copyfiles - Копіювання файлів з папки src/files у dist/files в режимі продакшн. Вмикач: true/false

3.3 version - Додавання унікальної версії до кінцевих CSS/JS файлів. Вмикач: true/false

3.4 hostname - Назва серверу

3.5 port - Порт серверу

4. html - Налаштування обробки HTML-файлів

4.1 beautify - Форматування в режимі продакшн

4.2 enable - Вмикач: true/false

4.3 indent - Налаштування типу відступів

5. styles - Налаштування роботи зі стилями та обробки SCSS/CSS-файлів

5.1 tailwindcss - Підключити Tailwindcss. Вмикач: true/false

5.2 pxtorem - Переводити PX в REM в режимі продакшн. Вмикач: true/false

5.3 critical - Створити та додати в HTML-файли критичні CSS-стилі. Вмикач: true/false

5.4 codesplit - Розділяти CSS-фали на окремі та підключати відповідним HTML-сторінкам в режимі продакшн. Вмикач: true/false

5.5 devfiles - Створення нестиснутих копій файлів в режимі продакшн. Вмикач: true/false

6. fonts - Налаштування обробки шрифтів

6.1 iconsfont - Створення шрифту з SVG іконок (Іконкового шрифту) та підключення його в проєкт. Вмикач: true/false

6.2 download - Завантажувати та робити локальними шрифти які підключені віддалено (наприклад з Google Fonts). Вмикач: true/false

7. images - Налаштування обробки зображень

7.1 svgsprite - Створення спрайту з SVG-іконок та підключення його в проєкт. Вмикач: true/false

7.2 optimize - Налаштування оптимізації зображень в режимі продакшн

7.3 enable - Вмикач: true/false. Якщо оптимізаця вимкнена зображення просто копіюються без конвертації та оптимізації

7.4 edithtml - Вносити зміни в HTML-файли. Якщо увімкнено, то HTML-файли будуть змінені згідно з налаштуваннями конвертації та створення додаткових розмірів зображень

7.5 sizes - Розміри файлів, які будуть додатково створені

7.6 dpi - Множники для створення додаткових файлів

7.7 attrignore - Атрибут тега IMG при наявності якого зображення не буде оброблено

7.8 modernformat - Налаштування конвертації в сучасні формати

7.9 enable - Вмикач: true/false

7.10 type - Формат зображення в який потрібно конвертувати: webp/avif

7.11 only - Залишати лише файли в сучасному форматі після конвертації. Вмикач: true/false

7.12 quality - Якість зображення

7.13 jpeg - Налаштування оптимізації зображень

7.14 quality - Якість зображення,

7.15 png - Налаштування оптимізації зображень

7.16 quality - Якість зображення

8. js - Налаштування обробки JavaScript-файлів

8.1 hotmodules - "Гаряче" підключення компонентів. Вмикач: true/false

8.2 devfiles - Створення нестиснутих копій файлів в режимі продакшн. Вмикач: true/false

8.3 bundle - Налаштування формування кінцевого файлу в режимі продакшн

8.4 enable - Система збирає весь код в один JS та один CSS файли незалежно від налаштування styles -> codesplit. Вмикач: true/false

9. react - Вмикає обробку React. Вмикач: true/false

10. vue - Вмикає обробку Vue. Вмикач: true/false

11. php - Налаштування PHP-серверу

11.1 enable - Вмикач: true/false

11.2 base - Шлях до папки де знаходяться файли PHP

11.3 hostname - Ім'я хосту

11.4 port - Порт

11.5 binary - Шлях до папки де встановлений інтерпритатор PHP

11.6 ini - Шлях до файлу налаштувань php.ini

12. pug - Налаштування препроцесору PUG

12.1 enable - Вмикач: true/false

13. ftp - Налаштування відправлення кінцевих файлів (з папки dist) на сервер (хостинг) по FTP в режимі продакшена

13.1 host - Ім'я FTP хосту

13.2 port - FTP Порт

13.3 remoteDir - Віддалена папка на сервері

13.4 user - Ім'я користувача FTP

13.5 password - Пароль користувача FTP

14 logger - Логи роботи модулів та компонентів збірки

14.1 terminal - Логи роботи модулів в терміналі. Вмикач: true/false

14.2 console - Логи роботи компонентів в консолі

14.3 enable - Вмикач: true/false

14.4 removeonbuild - Прибирає логи в режимі продакшн. Вмикач: true/false

15. projectpage - Cтворення сторінки проєкту в режимі продакшн. Сторінка буде містити посилання на усі HTML-файли проєкту для зручної демонстрації замовнику

15.1 enable - Вмикач: true/false

15.2 projectname - Назва проєкту, яка виводиться на сторінці замість %projectname%. Якщо порожньо, замінеться назвою папки проєкту

15.3 template - Шлях до шаблону сторінки проєкту

15.4 outfilename - І'я файлу. Якщо порожньо то створиться HTML-файл з назвою папки проєкту

16. aliases - Налаштування псевдонімів

17. coffee - Налаштування повідомлення про перерву

17.1 enable - Вмикач: true/false

17.2 text - Текст повідомлення

17.3 interval - Інтервал виводу повідомлення в хвилинах
