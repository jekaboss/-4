Статистика витраченого часу на проєкт

В стартовому шаблоні "Чертоги Фрілансера" 4 (далі ЧФ4) реалізована можливість збирати та переглядати статистику вистраченого часу на проєкт.

Статистика збирається під час роботи режиму розробника

Для того щоб почати збирати статистику необхідно увікнути цей функціонал в налаштуваннях ЧФ4:

statistics: {
enable: true,
...
}

Для перегляду статистики ви можете використовувати команду терміналу npm run stat, або увікнути автоматичний вивід при запуску режиму продакшн:

statistics: {
enable: true,
showonbuild: true
}

Наразі можна отримати такі дані:

1. Дата початку роботи над проєктом (перший запуск режиму розробника)

2. Дата кінця роботи над проєктом (остання зупинка режиму розробника, або останній запуск режиму продакшн)

3. Загальний час роботи над проєктом (час роботи режиму розробника)

4. Кількість сесій (кількість запусків/зупинок режиму розробника)
