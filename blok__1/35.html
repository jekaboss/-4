Завантаження на GitHub

В стартовому шаблоні "Чертоги Фрілансера" 4 (ЧФ4) є можливість автоматичного завантаження зібраного проєкту (папка dist) на сервіс GitHub.

Для цього вкажіть шлях до вашого репозиторію в налаштуваннях ЧФ4 git->repo, також є можливість вказати branch:

git: {
repo: "https://github.com/someuser/somerepo.git",
branch:"main"
},

Після налаштування ви можете користуватись командою термінала npm run git, яка об'єднує в собі команду збірки проєкту (режим продакшн) а також завантаження результату на GitHub.
